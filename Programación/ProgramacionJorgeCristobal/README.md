En este apartado he realizado con intellij en el lenguaje java una aplicacion que te permite logarte con usuarios que estan en un archivo llamado datos.xml que pueden ser
o bien administradores o usuarios y comprar productos de una lista de productos.

Los usuarios pueden comprar productos y cuando le dan a comprar la cantidad de stock baja tanto en la interfaz de javaFX como en el archivo datos.xml

Los administradores pueden o bien eliminar objetos o crear nuevos objetos que tambien serán actualizados tanto en la interfaz javaFX como en el archivo.xml

Las ventanas que hay son las de login, y luego la de administrador si entras con una cuenta que este vinculado al rol administrador y la ventana de usuario si entras con
un usuario que está vinculado a un rol usuario.

Cabe decir que como no he podido hacer el proytecto de bases de datos he decidio hacerlo con el .xml que tuvimos que hacer en lenguaje de marcas.
